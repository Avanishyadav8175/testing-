# Category Design Quick Reference

## Recommended Image Specifications

### Image Dimensions
- **Optimal**: 200x200 pixels
- **Minimum**: 150x150 pixels
- **Maximum**: 400x400 pixels
- **Aspect Ratio**: 1:1 (square)

### File Format
- **Best**: PNG with transparent background
- **Alternative**: SVG (vector graphics)
- **Acceptable**: JPG with white background

### File Size
- **Target**: 20-50 KB
- **Maximum**: 100 KB

## Design Style Guide

### Visual Style
- Simple, clean illustrations
- Flat design or minimal 3D
- Clear, recognizable icons
- Professional appearance

### Color Scheme
Match button colors to image theme:
- Government/Official: Navy Blue (#1e3a8a)
- Success/Approval: Green (#15803d)
- Results/Important: Red (#b91c1c)
- Alerts/Updates: Orange (#ea580c)
- Corporate: Purple (#6b21a8)
- Education: Violet (#7c3aed)
- Academic: Dark Blue (#0f172a)

### Tips
- Search for: "job icon", "government icon", "education illustration"
- Download PNG with transparent background
- Resize to 200x200px before uploading
- Optimize with TinyPNG or similar tools

## Upload Process
1. Prepare image (200x200px, PNG)
2. Go to Admin → Categories
3. Click "Add Category" or edit existing
4. Click "Choose File" under Category Image
5. Select your image
6. Wait for upload confirmation
7. Preview appears automatically
8. Save category

## Testing Checklist
- [ ] Image displays correctly in admin table
- [ ] Image shows on homepage category grid
- [ ] Button color matches design
- [ ] Hover effect works smoothly
- [ ] Mobile responsive (2 columns)
- [ ] Tablet responsive (3 columns)
